Restoring a human-first, corporate-free social web

RetroMesh — 2010s-style darknet overlay. No AI, no recommendation algorithms. Forked Tor Browser for access. Privacy tools optional.

Featured — Getting started

Early development: client (Tor Browser fork) + node daemon. Build and test in a sandboxed VM. See repository for instructions.

Get the repo Quick start

Quick start

1) Read warnings and local law. 2) Clone repo: git clone https://github.com/Retromesh/Retromesh.git. 3) See /client and /node READMEs. 4) Build in VM/sandbox before connecting to live networks.

Why RetroMesh

  • Social-first: groups, threads, events.
  • Human discovery: curated directories and peer invites.
  • Optional privacy: onion services, E2E messaging, ephemeral IDs.

Status

Early development — client & node work in progress. Use for testing and contribution.
